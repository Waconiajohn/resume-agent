# Snippets & APIs

Code patterns, API contracts, error postmortems, and reusable solutions.

## What goes here
- Code patterns that solved tricky problems and should be reused (e.g., React native value setter for E2E, abort signal pattern)
- API contract documentation for external services (Groq, Z.AI, Supabase, Perplexity)
- Bug postmortems with root cause analysis worth remembering
- Common error patterns and their fixes

## How to add
Use the [[Bug Postmortem]] template for bugs. For code patterns and API docs, create a plain note with a descriptive title.
