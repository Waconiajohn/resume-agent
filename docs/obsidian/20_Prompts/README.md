# Prompts

Reusable prompt patterns, system prompt snapshots, and A/B test results.

## What goes here
- System prompts that work well and should be preserved
- Prompt techniques that solved specific problems (e.g., getting structured JSON from Llama, reducing hallucination)
- A/B test results comparing prompt variants
- Model-specific quirk notes (e.g., Z.AI object coercion, Llama JSON comments)

## How to add
Use the [[Prompt Pattern]] template in `/Templates/`.
