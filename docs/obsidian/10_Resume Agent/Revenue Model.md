## Dual Revenue Engine

### Engine 1: SaaS Subscriptions

| Tier | Price | Margin | Purpose |
|------|-------|--------|---------|
| Free | $0/month forever, no CC | -$0.60-$1.00/user/mo | Mission + data + conversion funnel |
| Pro | $79/month ($790/year) | 90-96% | Main revenue driver |
| Concierge | $0 (with retirement planning conversation) | 38% | Funded by Engine 2 |
| Enterprise | $29-$49/seat/month | 85-92% | B2B outplacement |

Enterprise seat pricing: 25-50 seats at $49, 51-100 at $39, 101-250 at $29, 250+ custom.

### Engine 2: Financial Planner Referral Network

| Component | Rate |
|-----------|------|
| First-year commission | 20-25% of first-year advisory fee |
| Trailing commission (years 2-3) | 10% |
| After year 3 | Commission sunsets |
| Typical per-referral value | $1,000-$1,250 (on $500K AUM) |
| Conversion rate | 40-50% meeting-to-agreement |

Currently ~12 fiduciary planners, expanding to 100+. Projected 30-50% of total company revenue.

### Five-Step Handoff Protocol

1. User opt-in via Retirement Planning Agent
2. Planner matching (geography, asset level, specialization, personality fit)
3. Handoff document generation (career situation, asset range, concerns, retirement analysis, talking points)
4. Warm introduction email to both parties simultaneously
5. Follow-up tracking at 48 hours, 1 week, 2 weeks

### Lead Qualification Gates (5, before any referral)

1. Minimum $100K investable assets (self-reported)
2. Explicit user opt-in
3. Completed at least resume session + active job search
4. Geographic match to partner planner
5. Emotional readiness (Emotional Wellness Agent not flagging acute distress)

## LLM Cost Targets

| Metric | Target |
|--------|--------|
| Cost per agent session | $0.05-$0.25 |
| LLM cost as % of revenue | 3-6% at all scale points |
| Pro tier cost | $3-$8/user/month |
| Free tier cost | $0.60-$1.00/user/month |

## Key Pricing Decisions

- No a la carte agent pricing (agents are interdependent, fragments data advantage)
- No usage-based overage fees
- $79 positions as career intelligence system, not resume tool
- Homepage shows no pricing table — "Get started free. Upgrade when you're ready."
- Concierge uses "Apply" language to signal exclusivity

## The Flywheel

Free users improve AI -> Pro subscribers fund operations -> Concierge users generate premium revenue -> Success stories drive acquisition -> B2B adds batch scale -> AI gets smarter
